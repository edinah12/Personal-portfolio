Terravin SCM

This project was about building a supply chain management system for a wine production company. It was responsible for managing all activities from the time of acquiring raw materials upto delivery for example order processing, inventory, analytics, future sales prediction customer segmentation etc

My team and I built it using a laravel framework, bootstrap, CSS, php, python for machine learning models, MSQL for the database and javascript.
